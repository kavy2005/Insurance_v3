# Insurance Prediction System

An AI-powered insurance prediction application built using Python and Machine Learning.

## Features

- Predict insurance risk
- Analyze wearable data
- Parse medical reports
- Store user data in SQLite
- Interactive frontend

## Technologies Used

- Python
- Machine Learning
- SQLite
- Pandas
- Flask
- Scikit-learn

## Installation

```bash
git clone https://github.com/kavy2005/Insurance_v3.git

cd Insurance_v3

pip install -r requirements.txt

python app.py
```
